# Problem 4: Phase Difference

## Given
- Distance between two points: Δx = λ / 3

## Formula
Phase difference:  
Δφ = (2π / λ) * Δx

## Solution
Δφ = (2π / λ) * (λ / 3)  
Δφ = 2π / 3 radians

## Answer
- Phase difference: 2π / 3 radians

## Key Idea
- Phase difference is proportional to distance along the wave  
- Distance = fraction of wavelength → phase = same fraction of 2π
