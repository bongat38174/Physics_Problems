# ✅ **Problem 7: Standing Wave Modes**

### Given:

* String length: ( L = 80 ,\text{cm} = 0.8 ,\text{m} )
* Number of antinodes: 4

---

## 🔹 Step 1: Relation Between Wavelength and String

For a string with **n antinodes**:

[
L = n \frac{\lambda}{2} \quad \Rightarrow \quad \lambda = \frac{2L}{n}
]

---

## 🔹 Step 2: Substitute Values

[
\lambda = \frac{2 \times 0.8}{4}
]

[
\lambda = 0.4 ,\text{m}
]

---

## ✅ **Final Answer:**

[
\boxed{\lambda = 0.4 ,\text{m}}
]

---

## 💡 Key Idea:

* Each antinode corresponds to half a wavelength segment
* Wavelength decreases as the number of antinodes increases
