# ✅ **Problem 6: Wave Equation Analysis**

### Given Wave:

[
y(x,t) = 0.05 \sin(2\pi x - 50\pi t)
]

* (x) and (y) in meters, (t) in seconds

Compare with the general wave equation:

[
y(x,t) = A \sin(kx - \omega t)
]

---

## 🔹 Step 1: Amplitude

[
A = 0.05 ,\text{m}
]

---

## 🔹 Step 2: Wavelength

[
k = 2\pi \Rightarrow \lambda = \frac{2\pi}{k} = \frac{2\pi}{2\pi} = 1 ,\text{m}
]

---

## 🔹 Step 3: Frequency

[
\omega = 50\pi \Rightarrow f = \frac{\omega}{2\pi} = \frac{50\pi}{2\pi} = 25 ,\text{Hz}
]

---

## 🔹 Step 4: Wave Speed

[
v = f \lambda = 25 \times 1 = 25 ,\text{m/s}
]

---

## ✅ **Final Answers:**

* Amplitude: ( \boxed{0.05 ,\text{m}} )
* Wavelength: ( \boxed{1 ,\text{m}} )
* Frequency: ( \boxed{25 ,\text{Hz}} )
* Wave speed: ( \boxed{25 ,\text{m/s}} )

---

## 💡 Key Idea:

* Compare given wave with general form (y = A \sin(kx - \omega t))
* Extract (A), (k), and (\omega) to find all wave properties
