# Problem 3: Superposition Principle

## Given Waves
y1(x,t) = A sin(kx - ωt)  
y2(x,t) = A sin(kx + ωt)

## Using Identity
sin a + sin b = 2 sin((a+b)/2) cos((a-b)/2)

y(x,t) = 2A sin(kx) cos(ωt)

## Nodes
sin(kx) = 0 → x = n λ / 2, n = 0,1,2,…

## Answer
- Standing wave: y(x,t) = 2A sin(kx) cos(ωt)  
- Node positions: x = n λ / 2
