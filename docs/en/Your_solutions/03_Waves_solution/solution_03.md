# ✅ **Problem 3: Superposition Principle**

### Given Waves:

[
y_1(x, t) = A \sin(kx - \omega t), \quad y_2(x, t) = A \sin(kx + \omega t)
]

---

## 🔹 Step 1: Use the identity

[
\sin a + \sin b = 2 \sin\left(\frac{a+b}{2}\right)\cos\left(\frac{a-b}{2}\right)
]

Apply it to (y_1 + y_2):

[
y(x,t) = y_1 + y_2 = A\sin(kx - \omega t) + A\sin(kx + \omega t)
]

[
y(x,t) = 2A \sin(kx) \cos(\omega t)
]

---

## 🔹 Step 2: Identify Nodes

Nodes occur when (\sin(kx) = 0):

[
\sin(kx) = 0 \quad \Rightarrow \quad kx = n\pi
]

[
x = \frac{n \lambda}{2}, \quad n = 0,1,2,...
]

---

## ✅ **Final Answer:**

* **Standing wave equation:**
  [
  \boxed{y(x,t) = 2A \sin(kx)\cos(\omega t)}
  ]
* **Node positions:**
  [
  \boxed{x = \frac{n \lambda}{2}, \ n = 0,1,2,\dots}
  ]

---

## 💡 Key Idea:

* Superposition of two identical traveling waves moving in opposite directions → **standing wave**
* Nodes: points with zero displacement at all times
