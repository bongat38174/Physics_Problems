# ✅ **Solution 12: Work and Energy with a Constant Force**

---

## **Given:**

[
\vec F = (6,; 2)\ \text{N}, \quad m = 2\ \text{kg}
]
[
\vec v(0) = (1,; -1)\ \text{m/s}, \quad \vec r(0) = (0,;0)
]

---

## **1. Acceleration**

Using Newton’s second law:

[
\vec a = \frac{\vec F}{m}
]

[
\vec a = \left(\frac{6}{2},; \frac{2}{2}\right) = (3,;1)
]

---

## **2. Velocity**

[
\vec v(t) = \vec v_0 + \vec a t
]

[
\vec v(t) = (1,; -1) + (3,;1)t
]

[
\vec v(t) = (1 + 3t,; -1 + t)
]

---

## **3. Position**

[
\vec r(t) = \vec r_0 + \vec v_0 t + \frac{1}{2}\vec a t^2
]

[
\vec r(t) = (0,0) + (1,-1)t + \frac{1}{2}(3,1)t^2
]

[
\vec r(t) = \left(t + \frac{3}{2}t^2,; -t + \frac{1}{2}t^2\right)
]

---

## **4. Trajectory**

Eliminate (t):

From:
[
x = t + \frac{3}{2}t^2
]

[
y = -t + \frac{1}{2}t^2
]

This is a **parabolic trajectory** (curved motion in 2D under constant force).

---

## **5. Work done at (t = 3)**

Displacement:

[
\vec r(3) = \left(3 + \frac{3}{2}(9),; -3 + \frac{1}{2}(9)\right)
]

[
\vec r(3) = (16.5,; 1.5)
]

Work:

[
W = \vec F \cdot \vec r
]

[
W = (6,;2)\cdot(16.5,;1.5)
]

[
W = 6(16.5) + 2(1.5) = 99 + 3 = 102 \ \text{J}
]

---

## **6. Check Work-Energy Theorem**

Work-energy theorem:

[
W = \Delta K
]

Initial kinetic energy:

[
K_i = \frac{1}{2}m(v_x^2 + v_y^2)
= \frac{1}{2}(2)(1^2 + (-1)^2) = 2 \ \text{J}
]

Final velocity at (t=3):

[
\vec v(3) = (1+9,; -1+3) = (10,;2)
]

Final kinetic energy:

[
K_f = \frac{1}{2}(2)(10^2 + 2^2) = 104 \ \text{J}
]

[
\Delta K = 104 - 2 = 102 \ \text{J}
]

---

  ]

  ]

* Work-energy theorem is **verified** ✔️
